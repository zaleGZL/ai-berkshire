# MiniMax：三个被误读的数字

MiniMax（稀宇科技，港股 00100.HK）是"AI 六小虎"里第一家上市、也是最容易被看错的一家。

看错的地方，恰好是三个数字：一份"巨亏 18.7 亿美元"的财报，一个 67% 收入来自 C 端的身份，一个冲到 3 亿美元的 ARR。这三个数字，单看每一个都会把人带偏。这篇文章只做一件事——把它们拆开，看清底下真实的生意。

先把结论摆上桌：MiniMax 表面是家 C 端出海公司，利润表上其实是家 B 端 API 公司；它的模型只在第二梯队，但商业形态是六小虎里最完整的；而它和智谱 9 倍的市值差，恰好照出了它最深的那道软肋。

## 误读一：那份"巨亏 18.7 亿"的财报

MiniMax 2025 年的财报，头条数字是净亏损 18.72 亿美元。听着像在大出血。

但这里面约 16 亿，是"可转换优先股公允价值变动"——一个纯会计、不花现金的项目。原理反直觉：公司估值涨得越猛，这项账面亏损记得越大。早期投资人手里拿的是优先股，公司越值钱，这批优先股的账面价值被记得越高，而这部分增值在报表上要计成对普通股东的一笔负债。一分钱现金没花出去，纯粹是记账规则的纸面数字。而且这笔账在上市那一刻基本一笔勾销——优先股转成普通股，这个吓人的数字随之消失。

想看真实烧了多少钱，要看经调整净亏损——只有 2.5 亿美元。关键在于，这是在全年收入增长 159% 的情况下做到的：2025 年总收入 7904 万美元，同比 +158.9%，而真实亏损较上一年的 2.44 亿几乎没扩大。收入涨了一倍多，亏损没怎么涨，说明亏损率在快速收窄。

毛利率也从 2024 年的 12.2% 提到 25.4%，账上现金还有 10.5 亿美元。所以第一个误读要纠正：这家公司的报表看着吓人，经营其实在变好。

但 25.4% 这个整体毛利率，又藏着第二个、也是最重要的一个误读。

## 误读二：它不是 C 端公司，是 B 端公司

MiniMax 的收入分两块，拆开看，是两门完全不同的生意：

| 分部 | 2025 收入 | 占比 | 增速 | 毛利率 |
|---|---|---|---|---|
| AI 原生产品（C 端：Talkie、星野、海螺 App） | 5308 万美元 | 67.2% | +143% | 仅 4.7% |
| 开放平台/企业服务（B 端：MaaS、API） | 2596 万美元 | 32.8% | +198% | 69.4% |
| 合计 | 7904 万美元 | 100% | +159% | 25.4% |

看出问题了吗？占三分之二收入的 C 端，毛利率只有 4.7%，几乎白干；占三分之一的 B 端，毛利率高达 69.4%。

算一笔账：B 端毛利约 1800 万美元，而全公司总毛利才 2010 万美元。也就是说——B 端贡献了全公司约 90% 的毛利，却只占 33% 的收入。反过来，占 67% 收入的 C 端，只贡献了约 10% 的毛利。

所以 MiniMax 真实的样子是：C 端 App 负责走量、讲故事、攒用户和数据；B 端 API 负责赚钱。它表面是家出海陪伴公司，利润表上是家 B 端 API 公司。

更值得注意的是，这条真正赚钱的腿，还在加速：

| B 端（开放平台/企业服务） | 2024 | 2025 | 2025 Q4 单季 |
|---|---|---|---|
| 收入 | 872 万美元 | 2596 万美元 | 1054 万美元 |
| 占总收入 | 28.6% | 32.8% | 跃升至 41% |
| 毛利率 | 63.2% | 69.4% | 约 69% |
| 增速 | — | +198% | +278% |

B 端占比从前三季度的 29%，到第四季度单季直接跳到 41%；增速从全年的 198%，到 Q4 加快到 278%。新增收入几乎全靠这条腿在拉，而且是它质地最好的一条腿。

一句话：MiniMax 值不值钱，不看 C 端有多少用户，看这条 69% 毛利、翻倍增长的 B 端曲线，能不能在价格战里守住毛利、把占比从 41% 顶上 50%。

## 误读三：ARR 冲到 3 亿，不等于它有多强

2026 年 MiniMax 最亮眼的数字，是 ARR（年化经常性收入，即最新月收入乘以 12）一路狂飙：2 月突破 1.5 亿美元（创始人闫俊杰在业绩电话会披露），5 月冲到 3 亿美元、两个月翻倍（彭博社报道，随 A 股上市辅导披露）。券商测算年底可能到 8—10 亿美元。

数字很猛，但有两个坑，不填清楚就会高估它。

第一个坑：ARR 不是全年收入。ARR 是"当前这个月的挣钱速度 ×12"的瞬时值，天然比累计入账的全年收入大。5 月 ARR 已 3 亿，但分析师给 2026 全年确认收入才 2.2—3.0 亿美元（标普全球共识 2.19 亿、高盛 3.0 亿）。要提醒的是，官方并没有给 2026 收入指引，这些全年数字都是券商预测。谁把"ARR 3 亿"当成"2026 营收 3 亿"，就会高估。

第二个坑，也是关键：这 3 亿 ARR 是全公司总口径，含 C 端订阅，不是纯 API。这一点，在和智谱对比时会要命。

把两家最可比的港股 AI 公司并排看：

| 指标 | MiniMax | 智谱 GLM |
|---|---|---|
| 2025 收入 | 7904 万美元，+159% | 7.24 亿人民币（约 1.02 亿美元），+132% |
| 毛利率 | 25.4%（B 端 69%/C 端 4.7%） | 约 50%（整体更高） |
| 最新 ARR | 3 亿美元（总口径，含 C 端） | 17 亿人民币（约 2.5 亿美元，纯 API） |
| 2026 营收预测 | 2.2—3.0 亿美元 | 25—46 亿人民币（约 3.5—6.5 亿美元） |
| 市值（2026 年 7 月） | 约 910 亿港元（约 117 亿美元） | 约 8000 亿港元（约 1050 亿美元） |
| 业务重心 | C 端出海陪伴+视频+API | 企业级 MaaS/政企，国内为主 |

三个信号：

一，论基本面，智谱占优。2026 营收预测约是 MiniMax 的 1.5 到 2 倍，毛利率高一倍，而且智谱的 2.5 亿 ARR 是纯 API 口径——含金量比 MiniMax 掺了 C 端订阅的 3 亿更"经常性"。单论 B 端这门最值钱的生意，智谱大概是 MiniMax 的 2 到 4 倍。

二，但估值差（9 倍）远远超过基本面差（约 2 倍）。这多出来的溢价，买的不是当期业绩，是智谱 GLM-5.2 带来的"模型前沿定价权"叙事——智谱市销率一度到 1256 倍，那是信仰投票。

三，这恰好照出 MiniMax 的软肋：市场愿意为"模型领先"付高价，而模型领先，正是 MiniMax 最不占优的一环。

## 那 MiniMax 的模型，到底什么水平

顺着上面这句，把模型摆出来看：

- 综合能力第二梯队。最新的 M3 在 Artificial Analysis 综合榜得 44 分、全球第 25，落后头部（Claude、GPT、Gemini）10 到 16 分，连国产的智谱 GLM-5.2（51 分）都没追上。
- 语音是唯一强项。Speech 系列曾拿过全球第一，最新版仍在第一梯队，压过 ElevenLabs、OpenAI。
- 视频海螺掉队。旗舰 Hailuo 已跌出主流视频榜前列，被字节 Seedance、阿里、快手可灵、谷歌 Veo 全面反超，只剩"中游 + 便宜"。
- 靠低价换量。M2 系列是开放权重、价格只有 Claude 的 8%，在 OpenRouter 上做到编程类第二，但按美元算的收入只是中游——量大、价低、毛利薄。

一句话：性价比路线的第二梯队全能选手，语音最强，文本靠低价错位，纯推理和视频落后头部。

## 护城河，和最后的落点

把上面三个数字连起来，MiniMax 的护城河其实很清楚，也很尴尬：它没有一条又宽又深的河，壁垒是几个中等深度差异点的叠加——极致的成本效率（M1 训练成本仅约 53 万美元）、出海占比超 70%、多模态全栈、开放权重。这些能带来先发和规模优势，但每一条都能被复制，抵御不了巨头的资本和分发碾压。

而且它面临三重挤压：一，模型开源意味着客户零迁移成本，DeepSeek、智谱的免费策略会持续压它的 API 变现；二，阿里既是它最大的机构股东，又用通义千问正面竞争；三，C 端赖以生存的陪伴 App 踩在监管红线上，Talkie 已两度被下架。

所以落点是这样的：MiniMax 真正值钱、也真正该盯的，是那条 33% 占比、69% 毛利、翻倍增长的 B 端曲线——它是当期的利润和估值锚。C 端是流量和数据的护城河候选，B 端才是眼下赚钱的引擎。

它能不能成，取决于一件事：在护城河被开源和价格战填平之前，能不能把这条 B 端曲线转到自我造血。这是一场和时间赛跑的生意。

在一个模型三个月换一次王座的行业里，唯一不贬值的资产，是客户改不掉的习惯。而习惯，恰恰长在那条 69% 毛利的 B 端上。
